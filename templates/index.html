How Login Works

Our CRM system ensures secure and personalized access to each user. Here’s how it works:

1. Login Page

User enters their username and password to authenticate.

2. Role Redirection

Based on the user’s role, the system redirects them to the correct dashboard:

  • Employees → Employee Dashboard
  • Team Leads → Team Lead Dashboard
  • Admins → Admin Dashboard

3. Role-Specific Dashboard

Features and tools are displayed based on the user’s role.
